Box Model

Box model adalah konsep yang paling utama dari CSS Layouting, jika kalian paham mengenai ini maka akan mudah untuk kalian dalam melakukan pengaturan tata letak elemen pada website kalian.

  • Setiap elemen di halaman berada di halaman berada di dalam sebuah box
  • kita bisa mengatur ukuran dan posisi kotak tersebut
  • kita bisa memberi warna/gambar sebagai background kotak tersebut

ada 4 bagian komponen dalam box model:

  1. margin
  2. padding
  3. border
  4. konten
